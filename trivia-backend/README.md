# trivia-backend

Data processing and management for the trivia directory: scrapers, importers, Google Places enrichment, and the weekly event-occurrence scheduler. Data lives in Supabase (Postgres + PostGIS).

- **Production deployment and the cron scheduler:** see [DEPLOYMENT.md](./DEPLOYMENT.md).
